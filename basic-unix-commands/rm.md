# rm

这里需要注意的是在大多数 Unix 系统中 `rm` 命令相当于删除**并清空**，即使有办法找回也是要经历一番折腾，完全没有在 Windows 下双击垃圾桶，点击`还原文件`来的优雅。所以在执行 `rm` 命令时，一定要确认你当前的工作路径，以及你要删除的文件列表。

## 删除（多个）文件

这里加上 -f flag 为的是不让系统弹出提示，问你是否确认删除文件 `foo`. 

```bash
$ rm -f foo
$ rm -f foo bar a b c
```

## 删除文件夹

> 切勿执行 `rm -rf /`，血的教训 TAT

```bash
$ rm -rf test-dir/
```
